h1 até h6

Os elementos h1 ate o h6, são níveis de títulos e a partir do h2 em diante são subtítulos. Onde o h1 e o mais importante e o h6 o menos importante.

P

O elemento p é utilizado para criar conteúdo de texto, uma desrição para uma imagem.

Mark

A tag mark é utilizada para dar relevância a uma palavra ou uma frase expecífica do texto.

Small

A tag small cria um texto ou palavra com a fonte pequena. Mas lembrando que esse recurso não é mais recomendado.

i

A tag i deixa deixa uma parte de uma texto em itálico. Utiizada para destacar uma parte importante do texto.

u

A tag u, serve para para colocar um sublinhado embaixo de uma palavra ou frase para destacar informações importantes.

Strong

A tag strong deixa uma palavra com uma forte importância.

ol

A tag ol serve criar listas ordenadas.

ul

A tag ul cria lista não ordenada

li

A tag li são elementos que representa itens dentro de uma lista ordenada ou desodernada

a

A tag a é um elemento que cria links para navegar entre páginas no próprio site, ou pode levar para sites externos.

hr

A tag hr cria uma linha horizontal de separação de contéudo.

sub

A tag sub é usada para adicionar um texto subscrito e com uma fonte menor

sup

A tag sup ja é para deixar o texto sobrescrito.

blockquote

A tag blokquote é para criar um conteúdo de citação.

font

A tag font serva para alterar o tipo de fonte de um texto.

del

A tag del é utilizada para reprensatar uma parte do texto de foi excluído.

abbr

A tag abbr é utilizada para fazer um abrevição, e ela ja vem com o tittle para explicar a palavra que está sendo abreviada.
